# Form Validation

## Please note: 

**No data is captured in this project, this is not tied to any backend or database.  Feel free to enter any data you wish - the data will be logged in the console**  

## Thoughts

This is an incredibly simple project - form validation with HTML & CSS, without any need for JavaScript.  That being said still usinging JavaScript to check if passwords match.
