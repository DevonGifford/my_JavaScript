# Animated Tourism Page for Madrid, Spain - in progress...

This is a tourism landing page for the city of Madrid in Spain  
This projects main focus is to use scrolling animations solely with HTML. 

This website uses tailwind template from [Createive team website](https://www.creative-tim.com/learning-lab/tailwind-starter-kit/presentation)
